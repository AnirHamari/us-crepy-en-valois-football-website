## Deployment to GitHub Pages

### Step 1: Initialize Git Repository

```bash
cd uscv-football-website
git init
git add .
git commit -m "Initial commit: U.S. Crépy en Valois Football official website"
```

### Step 2: Create GitHub Repository

1. Go to [GitHub](https://github.com) and create a new repository named `us-crepy-en-valois-football-website`
2. Do NOT initialize with README, .gitignore, or license (we already have files)

### Step 3: Push to GitHub

```bash
git remote add origin https://github.com/YOUR_USERNAME/us-crepy-en-valois-football-website.git
git branch -M main
git push -u origin main
```

### Step 4: Enable GitHub Pages

1. Go to your repository on GitHub
2. Click on **Settings**
3. Scroll down to **Pages** section
4. Under "Source", select **main** branch
5. Click **Save**
6. Your site will be published at: `https://YOUR_USERNAME.github.io/us-crepy-en-valois-football-website/`
